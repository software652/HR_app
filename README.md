# HR Management App

A starter HR management app with a Next.js frontend and Express TypeScript API.

## Modules included
- Dashboard
- Employee management
- Leave management
- Attendance
- Payroll
- Recruitment
- Performance
- Reports
- Notifications
- Settings

## Run locally

```bash
npm install
npm run dev:api
npm run dev:web
```

Frontend: http://localhost:3000  
API: http://localhost:4000
